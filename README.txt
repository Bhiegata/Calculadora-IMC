
Calculadora de IMC
Este projeto é uma calculadora de IMC (Índice de Massa Corporal) desenvolvida com HTML, CSS e JavaScript. O objetivo é permitir que o usuário insira seu peso e altura para obter o valor do IMC e sua classificação de acordo com os padrões da OMS.

 
Estrutura do Projeto
calculadora-imc/
├── index.html          # Estrutura da página
├── index.css           # Estilos visuais
└── calculadora.js      # Lógica de cálculo do IMC



Como Usar
- Clone ou baixe este repositório.
- Abra o arquivo index.html em qualquer navegador moderno.
- Insira seu peso (em Kg) e altura (em metros).
- Clique no botão Calcular.
- O resultado do IMC e sua classificação serão exibidos na tela.


Fórmula do IMC
IMC = peso / altura * altura

 
	    Classificações

Faixa de IMC		Classificação
  
  Até 18.5		Abaixo do peso
 18.6 - 24.9		 Peso normal
 25.0 - 29.9		  Sobrepeso
 30.0 - 34.9	       Obesidade grau I
 35.0 - 39.9 	       Obesidade grau II
40.0 ou mais	       Obesidade grau III

Estilo

- Layout centralizado com bordas arredondadas e sombra.
- Gradiente de fundo com cores vibrantes.
- Placeholder estilizado e botão interativo com efeito hover.

Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)

Observações
- Há validação para evitar valores irreais (peso abaixo de 10kg ou altura acima de 2.5m).
- O IMC é arredondado para duas casas decimais.
